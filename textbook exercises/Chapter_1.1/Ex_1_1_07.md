# 1.1.07

## Problem

Give the value printed by each of the following code fragments:

```java
a.
    double t = 9.0;
    while (Math.abs(t - 9.0/t) > .001)
    t = (9.0/t + t) / 2.0;
    StdOut.printf("%.5f\n", t);
```

```java
b.
    int sum = 0;
    for (int i = 1; i < 1000; i++)
    for (int j = 0; j < i; j++)
    sum++;
    StdOut.println(sum);
```

```java
c.
    int sum = 0;
    for (int i = 1; i < 1000; i *= 2)
    for (int j = 0; j < N; j++)
    sum++;
    StdOut.println(sum);
```

## Solution

```
a. 3.00009
b. 499500
c. 100 (set N = 10)
```
